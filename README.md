# 💸 Personal Expense Tracker
A simple and intuitive web application for managing personal expenses, developed with HTML, CSS, and JavaScript (Vanilla).

It allows you to add, edit, delete, and filter expenses, with data persistence in the browser.

## 📌 Functionalities
* Add expenses (with description, value, category)
* Edit existing expenses
* Delete expenses
* Automatic summary (total spent / number of expenses)
* Filter by category
* Data persistence with localStorage data remains even after refreshing or closing the browser
* Dark / Light Mode theme switching with dynamic button
* Real-time clock and date (pt-PT format) 
* Weather API integration external API consumption (fetch) updated temperature and wind data

## 🧠 Applied Technical Concepts
* DOM manipulation 
* Events (addEventListener)
* Data Structures in JavaScript (arrays, objects)
* Functional Methods: map, filter, reduce
* Persistence with localStorage
* Asynchronous Programming (async/await, fetch)
* Separation of Responsibilities (well-defined functions) 
* Dynamic Interface Update (reactive UI without structures) 

## 🔮 Future Improvements

* Charts (e.g., Chart.js) 
* Backend storage (Firebase / Node.js) 
* Authentication system
